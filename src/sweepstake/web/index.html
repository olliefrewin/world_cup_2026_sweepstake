Quick Workflow
1
Before the tournament — collect Part 1 entries from everyone and upload them via the Submissions tab. Entries are not time-limited, but group picks score zero once the group stage has started.
2
Before the Round of 32 — collect and upload Part 2 bracket entries.
3
During the tournament — click Actuals → Refresh Now after matches to pull in the latest results. The leaderboard updates automatically.
4
After the final — set the Golden Boot winner in the Actuals tab, then export the final standings via Settings → Export Results as CSV.

Common Errors
Error messageWhat to do
"Participant X has already submitted"Click Remove P1 or Remove P2 in the Submissions list, then re-upload the corrected file.
"Validation errors"The spreadsheet contains an invalid team name or missing field. Ask the participant to fix it and resubmit.
"Could not classify file"The wrong file was uploaded, or the sheet name was changed. Make sure the original template was used.
Results not updating after RefreshThe data source may be lagging. Enter the result manually using the override key reference above.
"--" in leaderboard columnThat participant has not yet submitted that part. They appear at the bottom until both parts are uploaded.
